Start a new BABOK analysis project **in English (EN)**.

- Use `babok_new_project` with `language: EN` when MCP is available.
- Otherwise create the project under `projects/` with journal `language: EN`.
- Conduct Stage 0 (Project Charter) **in English**: ask about business trigger, sponsor, scope boundary, and Go/No-Go criteria — one question at a time.
